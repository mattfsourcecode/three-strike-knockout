# 🃏 Three-strike Knockout 🎉
 
## Tech stack:
### ✅  Tailwind CSS
### ✅  Matter.js
### ✅  jQuery
### ✅  Node.js
 
## Run it locally
 
### `yarn install && yarn start`

Then open [http://localhost:3000](http://localhost:3000) to start playing.
